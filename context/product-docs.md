# TechCorp Product Documentation

## Getting Started
1. Sign up at techcorp.com
2. Create your workspace
3. Invite team members via Settings > Team
4. Create your first project

## Password Reset
1. Go to login page
2. Click "Forgot Password"
3. Enter your email address
4. Check inbox for reset link
5. Link expires in 24 hours

## API Access
- API keys: Settings > Developer
- Rate limit: 1000 requests/hour
- Docs: docs.techcorp.com/api

## Billing & Plans
- Starter: $10/month (5 users)
- Pro: $25/month (25 users)
- Enterprise: Custom pricing
- Refunds: Contact billing team only

## Common Issues

### Cannot Login
- Check caps lock is off
- Clear browser cache
- Try incognito mode
- Reset password if needed

### App Not Working
- Check internet connection
- Refresh the page
- Log out and log back in
- Try different browser

### Data Not Syncing
- Check internet connection
- Refresh the page
- Log out and log back in
- Contact support if persists
